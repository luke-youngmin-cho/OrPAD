# OrPAD v1.0.0-beta.3

OrPAD v1.0.0-beta.3 completes the OrPAD identity update for Orchestration
Pipeline Authoring & Development.

## Changes

- Updated desktop app metadata, installer identity, protocol handling, web/PWA
  manifests, updater metadata, and release automation to use the OrPAD name.
- Refreshed in-app copy, documentation, security notes, tests, and localization
  files so user-facing surfaces use consistent OrPAD terminology.
- Rebuilt desktop, web, PWA, favicon, and installer icon assets with the OrPAD
  wordmark while preserving the logo aspect ratio.
- Published signed release manifests for Windows x64 and macOS universal
  release assets.
